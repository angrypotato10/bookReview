How to use BookReview's API

Make a request to route /api/isbn where isbn is an ISBN number. Website will return JSON response containing the book's title, author, publication date, ISBN number, review count and average score.

Resulting JSON will follow the format:

            
                {
                    "title": "1984",
                    "author": "George Orwell",
                    "year": 1949,
                    "isbn": "1632168146",
                    "review_count": 28,
                    "average_score": 5.0
                }
